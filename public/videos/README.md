# Video Assets

Place the following video files in this directory:

| Filename | Usage | Placement |
|---|---|---|
| `hero-loop.mp4` | Homepage hero background | Full-viewport, autoplay loop |
| `studio-portrait.mp4` | Homepage Studio section | Portrait (9:16), offset right |
| `shop-header.mp4` | Shop page header strip | Full-width, 30vh cinematic strip |
| `studio-tour.mp4` | About page tour section | Full-bleed parallax |

## Recommended specs
- Format: H.264 MP4 (broad compatibility)
- Max file size: 10MB per video (compress for web)
- Resolution: 1920x1080 (landscape) or 1080x1920 (portrait)
- No audio track needed (all videos are muted autoplay)
